# RunJavaScript -run javascript code in your code!
## How to install
npm:
```shell
$ npm install runjavascript
```
yarn:
```shell
$ yarn add runjavascript
```
## Example
javascript
```javascript
const runjavascript = require("runjavascript");

//run javascript code
console.log(runjavascript.run('console.log("hello!")'));

//resualt : hello!
```
